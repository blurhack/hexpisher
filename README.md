## Installation :

* `apt update`
* `apt install git -y`
* `git clone https://github.com/blurhack/hexpisher.git`
* `cd nexphisher`
#### > SETUP : `bash setup`
#### > SETUP [TERMUX] : `bash tmux_setup`
#### > Run : `bash nexphisher`

## Single Command :
```
apt update ; apt install git -y ; git clone git://github.com/blurhack/hexpisher.git ; cd hexphisher ; bash setup ; bash hexphisher
```
